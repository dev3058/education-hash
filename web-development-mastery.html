Overview

The Web Development course offered by Agumentik Educamp covers a range of fundamental web technologies, including HTML, CSS, JavaScript, jQuery, WordPress, and PHP frameworks such as Laravel and CodeIgniter. This course aims to equip participants with the skills needed to build dynamic and interactive websites.

What You will Learn?

  • Introduction to Web Technologies and HTML
  • CSS Styling and Layout
  • JavaScript Fundamentals
  • jQuery and DOM Manipulation
  • WordPress Development
  • PHP and MySQL Integration
  • Laravel Framework
  • CodeIgniter Framework
  • Project Development and Internship Preparation
  • Internship (2 months)
  • Weeks 1-2

    Introduction to Web Technologies and HTML

    • Lecture 1: Introduction to web development and the role of HTML
    • Lecture 2: HTML syntax and basic structure
    • Lecture 3: HTML tags and elements for content and structure
  • Weeks 3-4

    CSS Styling and Layout

    • Lecture 1: Cascading Style Sheets (CSS) fundamentals
    • Lecture 2: Selectors, properties, and values in CSS
    • Lecture 3: Layout techniques and responsive design principles
  • Weeks 5-6

    JavaScript Fundamentals

    • Lecture 1: Introduction to JavaScript and its role in web development
    • Lecture 2: JavaScript syntax and basic programming concepts
    • Lecture 3: Manipulating HTML elements and handling events with JavaScript
  • Weeks 7-8

    jQuery and DOM Manipulation

    • Lecture 1: Introduction to jQuery library and its features
    • Lecture 2: DOM traversal and manipulation with jQuery
    • Lecture 3: Social media advertising and audience targeting
    • Lecture 4: Event handling and animation using jQuery
  • Weeks 9-10

    WordPress Development

    • Lecture 1: Introduction to WordPress CMS
    • Lecture 2: Setting up a WordPress website
    • Lecture 3: Customizing themes, plugins, and templates
    • Lecture 4: Building dynamic websites with WordPress
  • Weeks 11-12

    PHP and MySQL Integration

    • Lecture 1: Introduction to PHP and server-side scripting
    • Lecture 2: Connecting PHP with MySQL database
    • Lecture 3: CRUD operations with PHP and MySQL
  • Weeks 13-14

    Laravel Framework

    • Lecture 1: Introduction to Laravel framework and its features
    • Lecture 2: Building web applications with Laravel
    • Lecture 3: Routing, model-view-controller (MVC) architecture, and database integration
  • Weeks 15-16

    CodeIgniter Framework

    • Lecture 1: Introduction to CodeIgniter framework and its advantages
    • Lecture 2: Developing web applications with CodeIgniter
    • Lecture 3:MVC architecture, database integration, and form validation
    • Lecture 3: Handling client requests, feedback, and objections
  • Weeks 17-18

    : Project Development and Internship Preparation

    • Lecture 1: Planning and designing a web development project
    • Lecture 2: Implementing acquired skills in a real-world scenario
    • Lecture 3: Preparing for the internship by building a portfolio
  • Weeks 19-20

    Internship (2 months)

    • Lecture 1: Applying the acquired skills in real-world projects
    • Lecture 2: Gaining hands-on experience in web development
    • Lecture 3: Collaborating with a team and working in a professional environment
